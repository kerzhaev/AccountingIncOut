## 3. Архитектурные решения (ADR)
*   Используем VbaModuleManager для управления модулями.
*   Кодировка файлов: Windows-1251 (кроме файлов .md их делай в стандартной кодировке)
*   GUI: UserForms (.frm) - UserFormVhIsh, UserFormProgress.
*   Логика отделена от представления (MVC).
*   Модульная архитектура с четким разделением ответственности.
*   Использование классов для обработки событий (AppEventHandler, WorksheetEventHandler).
*   Система логирования и резервного копирования для надежности.

### 3.1. Архитектура обработки событий
*   **TableEventHandler.bas** - Основной модуль обработки событий таблицы, контекстное меню, горячие клавиши
*   **AppEventHandler.cls** - Класс для обработки событий Application (WindowActivate, WindowDeactivate, WorkbookOpen)
  * Используется для управления горячими клавишами при активации/деактивации окна Excel
  * Инициализируется через TableEventHandler.InitializeTableEvents()
*   **WorksheetEventHandler.cls** - Класс для обработки событий листа (SelectionChange)
  * Используется для отслеживания изменений выделения на листе с таблицей

### 3.2. Стандарты кодирования
*   **Option Explicit** - Обязателен во всех модулях
*   **Типы данных** - Использование Long вместо Integer для индексов и счетчиков (кроме параметров событий MSForms)
*   **Обработка ошибок** - On Error GoTo ErrorHandler в каждой процедуре
*   **Документация** - JSDoc-стиль комментариев для всех функций и процедур

## 4. Карта модулей

### Основные модули (.bas):
*   **AutoStart.bas** - Автозапуск и инициализация системы
*   **DataManager.bas** - Управление данными таблицы, валидация, сохранение
*   **NavigationModule.bas** - Навигация по записям (первая, предыдущая, следующая, последняя)
*   **SearchModule.bas** - Мгновенный поиск по всем столбцам таблицы
*   **DashboardModule.bas** - Генерация и обновление dashboard с аналитикой
*   **HelperFunctions.bas** - Вспомогательные функции для работы формы
*   **RibbonCallbacks.bas** - Обработчики команд ленты Office
*   **CommonUtilities.bas** - Общие утилитарные функции (создан в результате рефакторинга 10.01.2025)

### Модули интеграции:
*   **ProvodkaIntegrationModule.bas** - Интеграция с 1С, массовое сопоставление проводок
*   **DoverennostMatchingModule.bas** - Мастер-сценарий доверенностей (слияние выгрузки в мастер + сопоставление с 1С, кэш в мастер-файле)
*   **ModuleWordLetters.bas** - Генерация писем Word по доверенностям

### Модули отчетов:
*   **ReportManager.bas** - Менеджер отчетов (базовый)
*   **StatusReportModule.bas** - Отчеты по статусам
*   **ServiceReportModule.bas** - Отчеты по службам
*   **DelayReportModule.bas** - Отчеты по задержкам
*   **ReportHelpers.bas** - Вспомогательные функции для отчетов

### Системные модули:
*   **SystemLogger.bas** - Журналирование операций системы
*   **SystemSettings.bas** - Управление настройками на скрытом листе
*   **SystemBackup.bas** - Резервное копирование
*   **ProgressManager.bas** - Управление прогресс-баром для длительных операций

### Модули обработки событий:
*   **TableEventHandler.bas** - Обработка событий таблицы, контекстное меню, горячие клавиши (Ctrl+E, Ctrl+D, Ctrl+Shift+M)
  * Инициализация системы интерактивных форм
  * Открытие формы для редактирования записи
  * Дублирование записей
  * Управление контекстным меню
*   **AppEventHandler.cls** - Класс для обработки событий Application (Excel 2021)
  * WindowActivate - активация горячих клавиш при активации окна
  * WindowDeactivate - деактивация горячих клавиш при деактивации окна
  * WorkbookOpen - автоматическая инициализация системы при открытии книги
*   **WorksheetEventHandler.cls** - Класс для обработки событий листа
  * SelectionChange - обновление статус-бара при изменении выделения

### Вспомогательные модули:
*   **SettingsModule.bas** - Модуль настроек (если есть)
*   **MdlBackupVBA.bas** - Резервное копирование VBA (если есть)

## 8. Важные замечания для разработчиков

### При работе с событиями
*   AppEventHandler инициализируется через TableEventHandler.InitializeTableEvents()
*   Не использовать AppEventHandler напрямую, только через TableEventHandler
*   Горячие клавиши управляются через AppEventHandler.WindowActivate/WindowDeactivate

### При добавлении новых модулей
*   Обязательно добавить Option Explicit первой строкой
*   Использовать Long вместо Integer для индексов и счетчиков
*   Добавить обработку ошибок (On Error GoTo ErrorHandler)
*   Документировать функции в формате JSDoc
