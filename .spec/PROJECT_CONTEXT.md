## 3. Архитектурные решения (ADR)
*   Используем VbaModuleManager для управления модулями.
*   Кодировка файлов: Windows-1251 (кроме файлов .md их делай в стандартной кодировке)
*   GUI: UserForms (.frm) - UserFormVhIsh, UserFormProgress.
*   Логика отделена от представления (MVC).
*   Модульная архитектура с четким разделением ответственности.
*   Использование классов для обработки событий (AppEventHandler, WorksheetEventHandler).
*   Система логирования и резервного копирования для надежности.

## 4. Карта модулей

### Основные модули (.bas):
*   **AutoStart.bas** - Автозапуск и инициализация системы
*   **DataManager.bas** - Управление данными таблицы, валидация, сохранение
*   **NavigationModule.bas** - Навигация по записям (первая, предыдущая, следующая, последняя)
*   **SearchModule.bas** - Мгновенный поиск по всем столбцам таблицы
*   **DashboardModule.bas** - Генерация и обновление dashboard с аналитикой
*   **HelperFunctions.bas** - Вспомогательные функции для работы формы
*   **RibbonCallbacks.bas** - Обработчики команд ленты Office
*   **CommonUtilities.bas** - Общие утилитарные функции (создан в результате рефакторинга 10.01.2025)

### Модули интеграции:
*   **ProvodkaIntegrationModule.bas** - Интеграция с 1С, массовое сопоставление проводок
*   **DoverennostMatchingModule.bas** - Трехпроходная система сопоставления доверенностей
*   **ModuleWordLetters.bas** - Генерация писем Word по доверенностям

### Модули отчетов:
*   **ReportManager.bas** - Менеджер отчетов (базовый)
*   **StatusReportModule.bas** - Отчеты по статусам
*   **ServiceReportModule.bas** - Отчеты по службам
*   **DelayReportModule.bas** - Отчеты по задержкам
*   **ReportHelpers.bas** - Вспомогательные функции для отчетов

### Системные модули:
*   **SystemLogger.bas** - Журналирование операций системы
*   **SystemSettings.bas** - Управление настройками на скрытом листе
*   **SystemBackup.bas** - Резервное копирование
*   **ProgressManager.bas** - Управление прогресс-баром для длительных операций

### Модули обработки событий:
*   **TableEventHandler.bas** - Обработка событий таблицы, контекстное меню
*   **AppEventHandler.cls** - Класс для обработки событий Application (Excel 2021)
*   **WorksheetEventHandler.cls** - Класс для обработки событий листа

### Вспомогательные модули:
*   **SettingsModule.bas** - Модуль настроек (если есть)
*   **MdlBackupVBA.bas** - Резервное копирование VBA (если есть)

**Примечание:** Module1.bas удален в результате рефакторинга 10.01.2025 (был устаревшим дубликатом DashboardModule.bas)

### UserForms (.frm):
*   **UserFormVhIsh.frm** - Основная форма ввода/редактирования документов
*   **UserFormProgress.frm** - Форма прогресс-бара для длительных операций
